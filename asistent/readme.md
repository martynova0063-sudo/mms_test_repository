# Агент автоматической регистрации в каталогах

Веб‑приложение для автоматической регистрации сайтов в онлайн‑каталогах с использованием Playwright.

## Функциональность

* заполнение форм регистрации на сайтах‑каталогах;
* парсинг данных с сайта (название, описание, логотип);
* обработка подтверждения email (заглушка, требует доработки);
* сохранение результатов в локальную базу данных SQLite;
* веб‑интерфейс для управления регистрацией;
* отображение истории регистраций в таблице.

## Требования

* Node.js 16+ (рекомендуется LTS‑версия);
* npm (идёт в комплекте с Node.js);
* свободное место на диске ~1 ГБ (для браузеров Playwright).

## Установка

### Установите зависимости:
npm install

### Загрузите браузеры для Playwright:
npx playwright install chromium --force
Эта команда скачает Chromium, Firefox и WebKit.

### Для работы с IMAP‑серверами:
npm install imap mailparser

### Создайте базу данных: 
node setup-database.js

### После выполнения в папке проекта появится файл registrations.db.

## Запуск проекта

### Запустите сервер:
node app.js

### Откройте браузер и перейдите по адресу: http://localhost:3000
